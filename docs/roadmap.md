# خارطة الطريق (Roadmap)

## نظرة عامة

JAAD BUILD يمر بـ 8 مراحل تطويرية تغطي جميع جوانب إدارة شركات المقاولات. كل مرحلة تبني على التي تسبقها لتقديم نظام ERP متكامل.
